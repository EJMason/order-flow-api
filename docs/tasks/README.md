# Implementation Tasks

Complete **TWO** tasks of your choice within **25 minutes**.

You may use AI tools (Cursor, Copilot, etc.) during implementation.

## Available Tasks

| Task | Description | Estimated Time |
|------|-------------|----------------|
| [Task A](./task-a-cancel-fulfillment.md) | Cancel Fulfillment endpoint | 10-15 min |
| [Task B](./task-b-write-tests.md) | Write unit tests for OrderService | 10-15 min |
| [Task C](./task-c-add-tracking.md) | Add tracking number endpoint | 8-12 min |
| [Task D](./task-d-order-summary.md) | Order summary endpoint | 10-12 min |
| [Task E](./task-e-validate-order.md) | Validate order before fulfillment | 8-10 min |

## Before You Start

1. Review the codebase structure
2. Run `npm run dev` to start the server
3. Run `npm run test:run` to verify tests pass
4. Pick your two tasks
